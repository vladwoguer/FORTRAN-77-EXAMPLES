# FORTRAN-77-EXAMPLES
A linguagem FORTRAN foi a primeira linguagem de alto nível para programação de computadores, tendo surgido na decada de 50.
FORTRAN 77 é a padronização ANSI, foi amplamente utilizado e ainda existe bastante código legado.
Tem formato fixo de 72 colunas
Sendo as 5 primeiras colunas destinadas à labels(se houver), a sexta coluna tem valor * se for continuação da anterior ou em branco
se for nova linha. O resto é destinado aos comandos da linguagem.

Compatível com Plato IDE 4.3.0
